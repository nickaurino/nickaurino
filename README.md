misinformation in short-form video. I built the data pipeline. Published at ICWSM 2026.
[Read the paper](https://workshop-proceedings.icwsm.org/abstract.php?id=2026_43) · [DOI](https://doi.org/10.36190/2026.43)

**📱 Dabble** — a hobby discovery app that nudges you off the feed and into actually trying
something. Built because boredom is where curiosity starts. *React Native · Expo · Firebase · iOS (App Store soon)*

**🏃 Cadence** — an iOS app that detects your walking or running cadence and matches music to
your pace, using a native Swift MusicKit module. *React Native · Expo · iOS*

**⚔️ CodeFades** — a real-time 1v1 competitive coding platform: Elo matchmaking, in-browser
editor, Docker-sandboxed execution, and WebSocket match sync. *Next.js · WebSockets · Docker*

**🧮 Logos** — a statically-typed programming language for encoding mathematical theorems,
built from scratch with sum/product types, lazy evaluation, and a package system.

**🛰️ Project Skywalker** — led avionics for a 2U CubeSat mission carrying an experimental MHD
